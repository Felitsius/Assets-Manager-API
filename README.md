# Asset Management API 🏢

Система управления имуществом предприятия, построенная на FastAPI, SQLAlchemy и PostgreSQL. Позволяет отслеживать и управлять имуществом компании, их перемещениями, обслуживанием и документацией.

## ✨ Возможности

### Основной функционал
- **Управление пользователями**: регистрация, аутентификация, разделение прав (Администратор/Пользователь)
- **Управление имуществом**: инвентаризация номерами, штрих-кодами и серийными номерами
- **Категории**: иерархическая структура для организации имущества
- **Местоположения**: многоуровневая структура
- **Перемещения**: отслеживание перемещений, выдач и возвратов с полной историей
- **Обслуживание**: планирование и запись ремонтов, осмотров, технического обслуживания

### Безопасность
- JWT-аутентификация
- Хеширование паролей (argon2)
- Разграничение доступа по ролям (Администратор/Пользователь)
- Защищенные эндпоинты с проверкой аутентификации

### Технические особенности
- Асинхронные операции с базой данных через SQLAlchemy
- Автоматическая документация API (Swagger/ReDoc)
- Миграции базы данных через Alembic
- Комплексная обработка ошибок
- Валидация входных данных через Pydantic
